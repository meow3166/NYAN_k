2) 깃 명령어 
* 빠져나갈떈 Q
• git init : 깃폴더 만들기
• git status : 상태보기
• git add 파일명 : 장전
• git  add . : 전체 장전 
• git config --global user.email "메일주소" 
• git config --global user.name "이름" 
• git config --list 
• git commit -m "커밋 메세지" 
• git log : 커밋 기록 
• git log --oneline : 커밋 기록 한줄씩
• git checkout 해시코드 (16진수 앞 7자리까지 또는 전체 _ 로그보면 나와있음 ) : 시간여행 (저장전으로) 
• git checkout - : 원위치 
• git branch -M main
• git remote add origin https://github.com/meow3166/NYAN_k.git
• git push -u origin main : 원격 리포지토리 주소설정 
• git push -u origin main : 업로드
