• ls -a
• touch 파일명 
• echo "출력 문자열" 
• echo "문자열" >> 파일명 
• cat 
• cd ..
• cd 폴더명 
• clear :
• rm 폴더명 또는 파일명 :걍 지우고 
• rm -rf 폴더명 또는 파일명 : 강제로 지우고
• mkdir 폴더이름 : 폴더 만들기
cp 파일명 파일명 : 파일 복사 
touch .gitignore

2) 깃 명령어 
* 빠져나갈떈 Q
• git init : 깃폴더 만들기
• git status : 상태보기
• git add 파일명 : 장전
• git  add . : 전체 장전 
• git log --graph --oneline :로그에 그래프 추가 
• git config --global user.email "메일주소" 
• git config --global user.name "이름" 
• git config --list 
• git commit -m "커밋 메세지" 
• git log : 커밋 기록 
• git log --oneline : 커밋 기록 한줄씩
• git checkout 해시코드 (16진수 앞 7자리까지 또는 전체 _ 로그보면 나와있음 ) : 시간여행 (저장전으로) 
• git checkout - : 원위치 
• git branch -M main
• git remote add origin https://github.com/meow3166/NYAN_k.git
• git push -u origin main : 원격 리포지토리 주소설정 
• git push -u origin main : 업로드
• git clone https://github.com/meow3166/NYAN_k.git (링크) .
 : 현재 폴더에 카피_원격 리포지토리에 갖고오는거 (!! . 을 빼면 폴더안 폴더 상황이 .. ) 
• git remote -v : 리포지토리 주소 출력 
• git remote remove origin : 리포지토리 주소 제거(초기화)
• git fetch : 원격지에 뭐가 있나 체크 
log를 봤을떄 origin/main 없으면 원격에서 뭐가 바꼇당. . !! 
• git pull : 원격에서 바꾼 후 반영 
•  git push -f origin main   : 강제변경..
